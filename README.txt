== Description ==

WooCommerce plugin for more options and customization. Simple and Easy to use.

[Documentation](https://en.condless.com/advanced-options-for-woocommerce/) | [Contact](https://en.condless.com/contact/)

= Features =
* **Quantity Processing**: Adds in the admin products table a column of the current items quantity in proccessing orders.

== Installation ==

= Minimum Requirements =
WordPress 5.2 or greater
PHP 7.0 or greater
WooCommerce 3.4 or greater

= Automatic installation =
1. Go to your Dashboard => Plugins => Add new
1. In the search form write: Condless
1. When the search return the result, click on the Install Now button

= Manual Installation =
1. Download the plugin from this page clicking on the Download button
1. Go to your Dashboard => Plugins => Add new
1. Now select Upload Plugin button
1. Click on Select file button and select the file you just download
1. Click on Install Now button and the Activate Plugin
